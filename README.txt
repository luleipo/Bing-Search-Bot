Mobile - Earns 100 points for mobile

1. install python
2. download bing-search-bot-mobile.py
3. set Chrome or Firefox to default web browser
4. use "User-Agent Switcher" Extension to switch to mobile
5. run bing-search-bot-mobile.py
________________________________________________________________________

2. download bing-search-bot-edge.py
3. run bing-search-bot-edge.py
